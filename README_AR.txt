تحديث المرحلة الثالثة: كشوف المعلمين والموظفين الرسمية
====================================================

طريقة التثبيت:
1) خذ نسخة احتياطية من المشروع.
2) فك ضغط محتويات هذا الملف داخل مجلد المشروع الرئيسي الذي يحتوي على frontend و backend.
3) اختر دمج المجلدات واستبدال الملفات المتطابقة.
4) أوقف Backend ثم شغله مجددًا: npm run dev
5) داخل المتصفح نفذ Ctrl + F5.

نطاق التعديل فقط:
- إضافة زري تصدير PDF وطباعة داخل تبويب عرض الموظفين والمعلمين.
- استخدام نفس القالب المدرسي الموحد الذي يعمل في كشف الطلاب.
- إضافة نافذة اختيار: كشف مختصر، كشف تفصيلي، أعمدة يدوية، حالة العاملين، وفئة التقرير.
- إضافة مسار Backend مخصص لكشوف العاملين.
- جعل قالب PDF المركزي يقبل تسمية العدد المناسبة (طلاب / معلمين / موظفين / عاملين).

ما لم يتم تعديله:
- تسجيل الموظف أو المعلم.
- الحفظ أو التعديل أو الحذف أو التفعيل والتعطيل.
- الصلاحيات أو قاعدة البيانات.
- واجهات أخرى خارج نظام الكشوف.

الاختبار:
1) افتح تسجيل الموظفين ثم تبويب العرض.
2) افتح تبويب المعلمين واضغط تصدير PDF.
3) تحقق من ظهور نافذة إعداد الكشف واختيار المعلمين افتراضيًا.
4) جرّب تنزيل كشف مختصر ثم كشف تفصيلي.
5) افتح تبويب الموظفين واضغط طباعة وتحقق من اختيار الموظفين افتراضيًا.
6) جرّب اختيار جميع العاملين من نافذة إعداد الكشف.
